5.3.1The Internet
‚ñº

The word 'Internet' is a shortened form of 'inter' and 'network' ‚Äî meaning interconnected networks. Think of the Internet as a network of networks ‚Äî the biggest WAN of all. Schools, organisations and homes all have networks that are part of the Internet.

The reach of the Internet is global ‚Äî all countries have cables, fibre-optic links and satellites that connect them. These links can be thought of as the Internet backbone ‚Äî the connections that link all networks together.

üî§ Domain Names

SUBJECT VOCABULARY

Domain name ‚Äî this is part of the URL for a resource on the Internet. When the domain name is used, it is converted to the correct IP address by the domain name service (DNS) and the connection takes place.

Domain names are more convenient and easier to remember than the four octets of binary numbers that make up an IP address.

e.g. The IP address of bbc.co.uk is 212.58.244.27 ‚Äî much harder to remember than the domain name!

üîÅ Domain Name Service (DNS)

This is an Internet service that translates domain names into IP addresses. The Internet is based on IP addresses, but domain names (being alphabetical) are easier to remember. Every time a user enters a domain name, a DNS must translate it into the corresponding IP address.

The DNS system is a network of servers. If one DNS server doesn't know how to translate a domain name, it asks another ‚Äî and so on ‚Äî until the correct IP address is returned to the user.

SUBJECT VOCABULARY

Internet service provider (ISP) ‚Äî an organisation that provides its customers with a connection to the Internet.

Router ‚Äî a piece of networking hardware used to forward packets of data from one network to another.

5.3.2Packets & Routing
‚ñº

The Internet uses the TCP/IP protocol stack to allow communication between all different networks. The IP part provides the technical rules on addressing ‚Äî giving each device a unique address to send data to/from.

When an Internet-connected computer wants to send data to another computer, the following steps happen:

1
Sending computer splits the data into packets
2
Each packet has a header containing: sender's address, destination address, current packet number, total number of packets
3
Packet sent to your ISP
4
Your ISP's router inspects the packet header ‚Äî decides where to send it based on the destination IP address
5
Packet forwarded through multiple routers until it reaches the destination network
6
Receiving computer puts packets back together. If they arrive out of order, the header information is used to reassemble them correctly
‚úèÔ∏è Exam tip: A router uses a routing table ‚Äî a list of rules about where to send packets for different destinations. A packet may travel through many routers before reaching its destination.
5.3.3The World Wide Web
‚ñº

The World Wide Web (WWW or 'the Web') is a service that runs on the Internet. It provides access to web pages linked together using a hypertext system. Hypertext is text containing links (hyperlinks) that take you to another document or web page. These documents can contain text, graphics and multimedia.

‚ö†Ô∏è Important: The Internet ‚â† the WWW. The Internet is the global infrastructure. The WWW is just one service that uses it (along with email, FTP, VoIP, etc.)

üåê Accessing the World Wide Web

You access the WWW using a program called a web browser. All browsers use the same protocols and the Internet to transfer information. The web browser's job is to convert the data received from a web server into a human-readable format.

üîÑ How the World Wide Web Works

SUBJECT VOCABULARY

HyperText Markup Language (HTML) ‚Äî essentially a text document that contains text to be displayed along with: details of how text should be formatted (font size, colour, etc.); details of any hyperlinks and where they link to using a URL; details of any objects such as pictures or videos to be shown in the web page.

1
User enters a URL (Uniform Resource Locator / web address) into a web browser. A URL may be for a web page or an individual file (e.g. a specific picture).
2
Computer uses the DNS to find the IP address of the required web server
3
Browser connects to the web server using the IP address and requests the relevant web page or object (picture, sound or video file)
4
Web page is transferred using HTTP or HTTPS
5
Data sent from server to browser is in HTML format. The browser displays the web page as described by the HTML

‚úèÔ∏è Exam tip: HTTP = unsecured transfer protocol. HTTPS = encrypted (secure) version. The 'S' stands for Secure and uses encryption to protect data in transit.

‚ñº

An IP address is a set of numbers that identify one particular computer. It is like a postal address ‚Äî it allows Internet data to be sent to the correct computer. It is therefore essential that all IP addresses conform to the same standard.

5.3.5Components Needed to Access the Internet
‚ñº

Devices are needed to connect to a LAN, and that LAN to a WAN such as the Internet. Devices can connect using cables or wirelessly using radio waves.

Figure 5.10 ‚Äî Devices allowing network users to connect to the Internet

üîå Switch
SUBJECT VOCABULARY

Switch ‚Äî a device that receives incoming data packets and redirects them to their destination on a local area network (LAN).

A switch links computers within a LAN. Cables from each computer feed into it so messages can be transmitted from one to another. Switches are 'intelligent' ‚Äî they can read the destination addresses and send them to only the intended computers rather than all of them. They build up a table of all the addresses on the network. This cuts down on network traffic.

üì∂ Wireless Access Point (WAP)
SUBJECT VOCABULARY

Wireless access point (WAP) ‚Äî a device that is used to connect one or more computers wirelessly to an existing physical wired network.

WAPs allow wireless devices to connect to a wired network using Wi-Fi. They convert data received through cables into a wireless signal and vice versa. Commonly used in public buildings to provide 'Internet hotspots'.

‚ö†Ô∏è WAPs are similar to switches but cannot direct messages to particular devices ‚Äî they transmit to all wireless devices within range.
üîÄ Router

Routers are similar to switches because they read address information, but they transmit messages between networks. A switch does this within a single network; a router does this across several networks.

‚Ä¢
Has a routing table ‚Äî a list of rules stating where to send packets for different destinations
‚Ä¢
When a packet arrives, the router looks at the destination IP address in the header and uses the routing table to decide where to forward it
‚Ä¢
A packet may be forwarded between several routers before reaching its destination
‚Ä¢
Commonly used in homes to share an Internet connection; can have both cable and Wi-Fi connections
e.g. When you view a web page hosted in the USA, data travels router to router until it reaches your network ‚Äî all in less than a second!
üì° Modem
SUBJECT VOCABULARY

Modem ‚Äî short for 'modulator-demodulator'. It modulates and demodulates signals (converts them from digital to analogue and vice versa) sent and received by a computer over a communications network.

A modem converts signals in a LAN (such as a home network) into signals that can travel along cables provided by the ISP. The type of modem depends on the cable type:

‚Ä¢
Copper telephone line ‚Äî carries electrical signals
‚Ä¢
Cable TV cable ‚Äî carries electrical signals
‚Ä¢
Fibre-optic cable ‚Äî carries signals as light

To make setup easier, modems are usually combined with routers in a single box ‚Äî referred to as an Internet router.

Switch ‚Äî within ONE network; sends to specific device only (builds address table) ‚úÖ
WAP ‚Äî within one network; broadcasts to ALL wireless devices üì°
Router ‚Äî BETWEEN networks; reads IP and forwards to correct network üîÄ
